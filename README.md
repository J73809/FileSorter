
# File Sorter (Terminal)

A simple terminal-based file sorter written in C# using **Spectre.Console**.
Sorts files into folders based on file type, with a small app-like TUI.

---

## Features

* Sorts files by type: images, videos, sounds, documents, archives, fonts, executables, scripts, 3D models, temporary files.
* Multi-selection of categories for sorting.
* Supports any folder path.

---

## How to Use

1. Clone or download the repository.
2. Open in your C# IDE (e.g., Visual Studio, VS Code).
3. Build and run the project.
4. In the terminal:

   * Enter the path to the folder you want to sort.
   * Select which file categories to sort.
5. Sorting complete! Files are organized into folders under the selected path.

---

## Requirements

* .NET 6+ or compatible runtime
* Spectre.Console NuGet package
